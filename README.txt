Get the latest version with:
  darcs get http://daimi.au.dk/~cgd/extensions/Reflect


To use this, please unpack and compile swig.

wget http://openengine.dk/data/libraries/swig.tgz
tar -vzxf swig.tgz
cd swig
./autogen.sh
./configure
make
